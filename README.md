# vim-notetaking

##### Vim으로 노트 작성하기
파일 이름은 `.note`로 끝나야 합니다.

##### Header
```
This is header:
```

##### Code block
```
`Some code.
 Another Code.`
```

##### `Ctrl+T` - 새로운 할 일 추가
```
- [ ] Dotfiles 푸시하기
```

##### `Ctrl+I` - 중요한 일 강조 표시하기
```
- [!] Dotfiles 푸시하기
```

##### `Ctrl+D` - 완료 상태로 표시하기
```
- [x] [2016-04-27] Dotfiles 푸시하기
```
